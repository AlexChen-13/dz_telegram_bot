Бот для перевода русского печатного текста на латиницу согласно указу: https://www.consultant.ru/document/cons_doc_LAW_360580/9eb761ae644ec1e283b3a50ef232330b924577cb/
Создан в рамках учебной программы по Elbrus DataScience
Инструкция к запуску через Docker:
1. Форкаете репозиторий на ваш аккаунт.
2. Скопируйте форк на ваш ПК
3. Соберите докер образ: 

  docker build -t ваш_образ .
  
4. При запуске контейнера вам, понадобится указать переменные окружения, такие как токен вашего Telegram-бота:

  docker run -d --name ваш_контейнер -e TELEGRAM_TOKEN=ваш_токен -p 8080:8080 ваш_образ

- Замените `ваш_контейнер` на имя, которое вы хотите задать вашему контейнеру.
- Замените `ваш_токен` на токен вашего Telegram-бота.
- Порт `8080` можно заменить на другой, если ваш бот использует другой порт.

5. Чтобы убедиться, что контейнер работает можете использовать следующую команду:

  docker ps -a
